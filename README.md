# Task 1 - Understanding Dataset & Data Types

## Objective

To understand the structure of a dataset and identify different data types using Pandas.

## Dataset Information

* Rows: 5300
* Columns: 12

## Tools Used

* Python
* Pandas
* NumPy
* Jupyter Notebook

## Tasks Performed

1. Loaded the dataset using Pandas.
2. Checked dataset shape and dimensions.
3. Identified numerical and categorical variables.
4. Examined data types using info().
5. Generated summary statistics using describe().
6. Checked for missing values.
7. Explored unique values in categorical columns.

## Key Columns

* Age
* Gender
* Profession
* Marital_status
* Education
* No_of_Dependents
* Personal_loan
* House_loan
* Partner_working
* Salary
* Partner_salary
* Total_salary

## Outcome

Successfully analyzed the dataset structure and understood the different data types present in the dataset. This task helped build a foundation for Exploratory Data Analysis (EDA).
